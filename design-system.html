Typography

We use a system font stack for performance and native feel across all devices. The typographic scale ensures readability and clear hierarchy.

Heading 1 (.text-4xl .font-bold)

Used for primary screen titles.

Heading 2 (.text-2xl .font-bold)

Used for major section titles like the balance card amount.

Heading 3 (.text-lg .font-semibold)

Used for sub-section titles like 'Rewards' or 'News'.

Body Text (.text-base)

Default text size for most content.

Subtle Text / Labels (.text-sm)

Used for captions, icons labels, and less important information.

Extra Small Text (.text-xs)

Used for footnotes and very fine print.

Layout & Spacing

Our layout is mobile-first, contained within a max-width of 420px to simulate a mobile screen. Spacing is based on a 4px grid system (Tailwind's default scale).

  • Container: The main app wrapper is centered with `max-w-sm` (384px) or `max-w-md` (448px) and a subtle shadow.
  • Padding: Main content areas use `p-4` (16px) for consistent internal spacing.
  • Gaps: Flexbox and Grid layouts use `gap-4` (16px) for spacing between elements.
  • Margins: Sections are separated by `mb-6` (24px).
